Placebo

To use Placebo, you can:

  1. Fork & clone the repository in your working directory
  2. Open the HTML files in your browser, under file-protocol
  3. Use preset commands to build/minify assets (coming soon)
  4. Include the libraries of your choice — we haven't included any
  5. Use the included app.js to quickly kick-off a Node.js server
